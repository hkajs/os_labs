To compile ResourceAllocater.java enter:

cd src
javac ResourceAllocator.java

Add the input txt files to the src folder.
Then after the program is compiled run the program by entering:

java ResourceAllocator filename.txt

Some examples:

java ResourceAllocator input-01.txt
java ResourceAllocator input-05.txt
java ResourceAllocator input-10.txt
